## Features

- 🔐 Notes and any metadata are 100% end-to-end encrypted
- 🏖 Offline mode
- ☁️ Zero trust cloud sync (_self-hosted cloud sync coming soon!_)
- 🙅‍♂️ No telemetry/analytics/data collection
- 🌙 _Dark mode coming soon!_
